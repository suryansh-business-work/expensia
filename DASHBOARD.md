### 1. **Total Till Now**

### 2. **Date Filters**
   - **Today**: Current day expenses
   - **Yesterday**: Previous day expenses
   - **Last 7 Days**: Weekly overview
   - **Last Month**: Previous month's data
   - **This Month**: Current month expenses
   - **India FY**: Indian Financial Year (April 1 - March 31)
   - **This Year**: Calendar year (January - December)
   - **Custom Date**: Select any date range

### 3. **Summary Statistics**
   - **Total Expenses**: Sum of all expenses in selected period
   - **Average Expense**: Mean transaction value
   - **Total Transactions**: Number of expense entries

### 4. **Visual Charts**

   #### Bar Chart - Expenses by Category
   - Horizontal comparison of spending across categories
   - Easy to identify highest spending categories

   #### Pie Chart - Category Distribution
   - Visual percentage breakdown
   - Color-coded for easy identification
   - Interactive tooltips

   #### Line Chart - Monthly Trend
   - Shows expense patterns over 12 months
   - Year selector to view historical data
   - Helps identify spending trends

### 5. **Category Breakdown**
   - Detailed list of all categories
   - Shows transaction count per category
   - Total amount spent in each category
   - Sorted by highest to lowest spending
